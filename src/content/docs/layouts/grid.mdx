## 基本的な使い方
`data-grid` 属性を指定するとグリッドコンテナーとなり、子要素たちがグリッドアイテムになります。




<iframe src="/olayout-docs/demo/grid/demo-1.html" style="min-height: 17rem" ></iframe>



```html "data-grid"
// index.html
<div data-grid>
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

### 表示
`data-grid` 属性はデフォルトでブロックレベルのグリッドコンテナーになります。
レスポンシブなどで、グリッドコンテナーではなくしたい時には、`unset` を指定することでグリッドコンテナーでなくすこともできます。


```html "unset"
// index.html
<div data-grid="unset">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

### 自動配置
`data-grid` 属性には、自動配置されるグリッドアイテムがどのようにグリッドに配置されるかを指定することができます。
デフォルトは `row` となっており、必要に応じて新しい行を自動で生成しグリッドアイテムを配置します。そのため、行方向<small>（列があれば Z 型）</small>に伸びていくグリッドになります。

`col` を指定すると、必要に応じて新しい列を生成してグリッドアイテムを配置するように変更できます。つまり、列方向<small>（行があれば N 型）</small>に伸びるグリッドにできます 。

<iframe src="/olayout-docs/demo/grid/demo-flow-1.html" style="min-height: 9rem; height: 9rem;" ></iframe>


```html "col"
// index.html
<div data-grid="col">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```


### 列の生成
`data-grid`属性に `cols-0` ~ `cols-12` を値を使って列（最大12列）を明示的に作成できます。
列数を超えた要素は、[自動配置](#自動配置) の設定に従って自動で生成される列か行に配置されます。


<iframe src="/olayout-docs/demo/grid/demo-cols-1.html" style="min-height: 13rem"></iframe>



```html "cols-3"
// index.html
<div data-grid="cols-3" data-gap="3">
  <div>01</div>
  <div>...</div>
  <div>06</div>
</div>
```


### 行の生成
`data-grid`属性に `rows-0` ~ `rows-12` を値を使って行（最大6行）を明示的に作成できます。
行数を超えた要素は、[自動配置](#自動配置) の設定に従って自動で生成される列か行に配置されます。

<iframe src="/olayout-docs/demo/grid/demo-rows-1.html" style="min-height: 16rem"></iframe>



```html "rows-3"
// index.html
<div data-grid="col rows-3" data-gap="3">
  <div>01</div>
  <div>...</div>
  <div>06</div>
</div>
```



### 間隔

`data-gap` 属性と組み合わせることで、各アイテム感の間隔を指定できます。


<iframe src="/olayout-docs/demo/grid/demo-gap-1.html"></iframe>



```html "data-gap="3""
<div data-grid="cols-3" data-gap="3">
  <div>01</div>
  <div>...</div>
  <div>06</div>
</div>
```


`data-gap` 属性は、フレックスコンテナーだけの属性ではないため、このページでの説明を割愛します。
間隔の詳細は、[data-gap属性のドキュメント](/olayout-docs/layouts/gap/)を参照してください。



### 位置

`data-place`属性と組み合わせることで、自由にグリッドアイテムの位置を揃えることができます。


<iframe src="/olayout-docs/demo/grid/demo-place-1.html" style="min-height: 15rem; height: 15rem"></iframe>


```html "data-place="+middle""
// index/html
<div data-grid="cols-3" data-place="+middle" data-gap="3">
  <div>01</div>
  <div>...</div>
  <div>06</div>
</div>
```

`+center` や `+middle` を使えば、要素をエリアのど真ん中に配置することもできます。


<iframe src="/olayout-docs/demo/grid/demo-place-2.html" style="min-height: 20rem;"></iframe>


```html "+center" "+middle"
// index/html
<div data-grid data-place="+center">
  <div>...</div>
</div>
```

`data-place` 属性は、フレックスコンテナーだけの属性ではないため、このページでの説明を割愛します。
順番の詳細は、[data-place属性のドキュメント](/olayout-docs/layouts/place/)を参照してください。

## レスポンシブ対応

各属性の値を指定する前に、ブレイクポイント名と:を付けることで、ブレイクポイントごとにスタイルを設定できます。

例えば、スマートフォンでは1列、タブレットは2列、PCでは3列にしたいということもあるでしょう。

そのような、ある一定のブラウザサイズから列数が変わるような設定も簡単です。



<iframe src="/olayout-docs/demo/grid/demo-breakpoint-cols-1.html" style="min-height: 13rem"></iframe>


```html "xs:cols-2" "sm:cols-3"
<div data-grid="xs:cols-2 sm:cols-3" data-gap="3">
  <div>01</div>
  <div>...</div>
  <div>06</div>
</div>
```

## 特定の箇所だけの値

特定の箇所のみに利用する列や行がある場合は、カスタプロパティを直接上書きすることで対応できます。

| カスタムプロパティ名 | 値  |
| -------------------- | --- |
| `--ol-grid-cols` |  `grid-template-columns` の値  |
| `--ol-grid-cols` |   `grid-template-rows` の値   |
| `--ol-grid-auto-cols` |   `grid-auto-columns` の値   |
| `--ol-grid-auto-rows` |   `grid-auto-rows` の値   |


### 手軽に任意の値を使う

手軽な方法は、`style` 属性を使ってカスタムプロパティを上書きすることです。

これを使えば、`auto-fill` や `auto-fit` といったCSSの値を活用した、メディアクエリを使わないレスポンシブなグリッドレイアウトを生成するテクニックを利用することもできます。

<iframe src="/olayout-docs/demo/grid/demo-custom-cols-1.html"></iframe>


```html "--ol-grid-cols"
// index.html
<div data-grid style="--ol-grid-cols: repeat(auto-fill, minmax(120px, 1fr))" data-gap="3">
  <div>01</div>
  <div>...</div>
  <div>06</div>
</div>
```

この方法はブレイクポイントごとに値を変更することができません。

### ブレイクポイントごとの値

ブレイクポイントごとに値を変更したい場合は、その要素に`class`属性かカスタムデータ属性を使って任意の値を指定し、そのセレクターに対してCSSファイル内でメディアクエリを利用して変更してください。

例えば、これを使って `data-grid`属性に元々は用意されていない、暗黙的に生成される列や行の大きさを指定することができます。

:::note
`data-grid`属性には、CSSの `grid-auto-columns`や `grid-auto-rows`を変更する値は用意していませんが、`--ol-grid-auto-cols` や `--ol-grid-auto-rows` を使えば変更できます。
:::

<iframe src="/olayout-docs/demo/grid/demo-custom-rows-1.html" style="min-height: 19rem"></iframe>



```html "my-auto-rows"
// index.html
<div data-grid="cols-3 my-auto-rows" data-gap="3" >
  <div class="box">01</div>
  <div class="box">...</div>
  <div class="box">06</div>
</div>
```

```css
// style.css
[data-grid~="my-auto-rows"] {
  --ol-grid-auto-rows: minmax(100px, auto);
}

@media (--ol-md) {
  [data-grid~="my-auto-rows"] {
    --ol-grid-auto-rows: max-content;
  }
}

@media (--ol-lg) {
  [data-grid~="my-auto-rows"] {
    --ol-grid-auto-rows: minmax(0, 1fr);
  }
}
```
