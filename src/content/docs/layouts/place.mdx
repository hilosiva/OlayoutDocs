## 基本的な使い方

`data-place`属性は、主に `data-flex` 属性（フレックスコンテナー）、や `data-grid` 属性（グリッドコンテナー）と組み合わせて利用します。

### 主軸/水平軸のコンテンツ全体の位置



フレックスコンテナーの主軸方向やグリッドコンテナーの水平軸方向のコンテンツ全体の位置を、`start` 、`center` 、`end` や `between` で指定できます。


:::note
フレックスコンテナーの主軸は、`data-flex` 属性の値が `row` または `row-reverse` の場合は水平軸、 `col` または `col-reverse` の場合は垂直軸となります。
:::



<iframe src="/olayout-docs/demo/place/demo-flex-1.html" style="min-height: 120px"></iframe>


```html "data-place="center""
// index.html
<div data-flex data-gap="2" data-place="center">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

### 交差軸/垂直軸の全アイテムの位置



フレックスコンテナーの交差軸方向やグリッドコンテナーの垂直軸方向の全アイテムの位置揃えは、`first` 、`middle` 、`last` や `stretch` で指定できます。


:::note
フレックスコンテナーの交差軸は、`data-flex` 属性の値が `row` または `row-reverse` の場合は垂直軸、 `col` または `col-reverse` の場合は水平軸となります。
:::



<iframe src="/olayout-docs/demo/place/demo-flex-2.html" style="min-height: 120px"></iframe>


```html "middle"
// index.html
<div data-flex data-gap="2" data-place="middle">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

### 水平軸と垂直軸のコンテンツ全体の位置



フレックスコンテナーの主軸と交差軸、グリッドコンテナーの水平軸と垂直軸の各方向のコンテンツ全体の位置を、`+start` 、`+center` 、`+end` や `+between` で一括指定できます。



<iframe src="/olayout-docs/demo/place/demo-place-1.html" style="min-height: 320px"></iframe>


```html "+center"
// index.html
<div data-grid data-place="+center" data-height="device">
  <div>...</div>
</div>
```

### 水平軸と垂直軸の全アイテムの位置



フレックスコンテナーの交差軸、グリッドコンテナーの水平軸と垂直軸の各方向のコンテンツ全体の位置を、`+start` 、`+center` 、`+end` や `+between` で一括指定できます。

:::note
フレックスコンテナーの主軸には適用されません
:::


<iframe src="/olayout-docs/demo/place/demo-place-2.html" style="min-height: 320px"></iframe>


```html "+middle"
// index.html
<div data-grid data-place="+center +middle" data-height="device">
  <div>...</div>
  <div>...</div>
</div>
```

## レスポンシブ対応

各属性の値を指定する前に、ブレイクポイント名と:を付けることで、ブレイクポイントごとにスタイルを設定できます。


<iframe src="/olayout-docs/demo/place/demo-breakpoint-1.html" style="min-height: 160px"></iframe>



```html "sm:end" "sm:middle"
// index.html
<div data-flex data-place="center stretch sm:end sm:middle" data-gap="2">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```



## 特定の箇所だけの値

特定の箇所のみ順序を変更したいなど、一部だけ利用したいスタイルがある場合は、カスタプロパティを直接上書きすることで対応できます。

| カスタムプロパティ名 | 値  |
| -------------------- | --- |
| `--ol-align-content` |   `align-content` の値  |
| `--ol-justify-content` |   `justify-content` の値  |
| `--ol-align-items` |   `align-items` の値  |
| `--ol-justify-items` |   `justify-items` の値  |


### 手軽に任意の値を使う

手軽な方法は、`style` 属性を使ってカスタムプロパティを上書きすることです。



<iframe src="/olayout-docs/demo/place/demo-custom-1.html" style="min-height: 8rem;"></iframe>


```html "--ol-justify-content: space-around"
// index.html
<div data-flex data-place data-gap="2" style="--ol-justify-content: space-around">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

この方法はブレイクポイントごとに値を変更することができません。

### ブレイクポイントごとの値

ブレイクポイントごとに値を変更したい場合は、その要素に`class`属性かカスタムデータ属性を使って任意の値を指定し、そのセレクターに対してCSSファイル内でメディアクエリを利用して変更してください。


<iframe src="/olayout-docs/demo/place/demo-custom-2.html" style="min-height: 8rem"></iframe>



```html "my-place"
// index.html
<div data-flex data-place="my-place" data-gap="2">
  <div>01</div>
  <div>02</div>
  <div>03</div>
</div>
```

```css
// style.css
[data-place~="my-place"] {
  --ol-justify-content: space-evenly;
}

@media (--ol-md) {
  [data-place~="my-place"] {
    --ol-justify-content: space-around;
  }
}

@media (--ol-lg) {
  [data-place~="my-place"] {
    --ol-justify-content: space-between;
  }
}
```
