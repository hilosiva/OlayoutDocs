## 基本的な使い方
グリッドアイテムに、`data-grid-item` 属性を指定すると、グリッド内のどこにどのように配置するかの設定ができます。




### 列の結合
`data-grid-item` 属性に `col-1` から `col-12` の値を使って結合する列の数を指定できます。また、`col-full` を指定すると、その行のすべての列を結合します。


<iframe src="/olayout-docs/demo/grid-item/demo-col-1.html" style="min-height: 20.5rem;" ></iframe>


```html "col-2" "col-3" "col-full"
// index.html
<div data-grid="cols-4" data-gap="2">
  <div>01</div>
  <div data-grid-item="col-2">02</div>
  <div>03</div>
  <div data-grid-item="col-full">04</div>
  <div data-grid-item="col-3">05</div>
  <div>06</div>
</div>
```

### 列の開始位置
`data-grid-item` 属性に `col-start-1` から `col-start-12` の値を使ってアイテムの配置を開始する列のグリッド線の番号を指定できます。

<iframe src="/olayout-docs/demo/grid-item/demo-col-2.html" style="min-height: 20.5rem;" ></iframe>


```html "col-start-1" "col-start-2" "col-start-3" "col-start-4"
// index.html
<div data-grid="cols-4" data-gap="2">
  <div data-grid-item="col-start-2 col-2">01</div>
  <div data-grid-item="col-start-1">02</div>
  <div data-grid-item="col-start-4">03</div>
  <div>04</div>
  <div data-grid-item="col-start-3">05</div>
  <div>06</div>
</div>
```

### 列の終了位置
`data-grid-item` 属性に `col-end-1` から `col-end-12` 値を使ってアイテムの配置を終了する列のグリッド線の番号を指定できます。`col-start-*` と組み合わせて列を結合することもできます。

<iframe src="/olayout-docs/demo/grid-item/demo-col-3.html" style="min-height: 20.5rem;" ></iframe>


```html "col-end-5" "col-end-2" "col-end-3" "col-end-4"
// index.html
<div data-grid="cols-4" data-gap="2">
  <div data-grid-item="col-end-4 col-2">01</div>
  <div data-grid-item="col-end-3">02</div>
  <div data-grid-item="col-start-3 col-end-5">03</div>
  <div data-grid-item="col-end-4 col-3">04</div>
</div>
```



### 行の結合
`data-grid-item` 属性に `row-1` から `row-12` の値を使って結合する行の数を指定できます。また、`row-full` を指定すると、その行のすべての列を結合します。


<iframe src="/olayout-docs/demo/grid-item/demo-row-1.html" style="min-height: 19.5rem;" ></iframe>


```html "row-2" "row-3" "row-full"
// index.html
<div data-grid="cols-3" data-gap="2" style="--ol-grid-auto-rows: 1fr">
  <div>01</div>
  <div data-grid-item="row-2">02</div>
  <div data-grid-item="row-3">03</div>
  <div data-grid-item="row-2">04</div>
  <div>05</div>
</div>
```

### 行の開始位置
`data-grid-item` 属性に `row-start-1` から `row-start-12` の値を使ってアイテムの配置を開始する行のグリッド線の番号を指定できます。

<iframe src="/olayout-docs/demo/grid-item/demo-row-2.html" style="min-height: 19.5rem;" ></iframe>


```html "row-start-1" "row-start-2" "row-start-3" "row-start-4"
// index.html
<div data-grid="rows-3 col" data-gap="2">
  <div data-grid-item="row-start-2">01</div>
  <div>02</div>
  <div>03</div>
  <div data-grid-item="row-start-3">04</div>
  <div data-grid-item="row-start-2">05</div>
  <div data-grid-item="row-start-1">06</div>
</div>
```

### 行の終了位置
`data-grid-item` 属性に `row-end-1` から `row-end-12` 値を使ってアイテムの配置を終了する行のグリッド線の番号を指定できます。`row-start-*` と組み合わせて列を結合することもできます。

<iframe src="/olayout-docs/demo/grid-item/demo-row-3.html" style="min-height: 19.5rem;" ></iframe>


```html "row-end-4" "row-end-3"
// index.html
<div data-grid="rows-3 col" data-gap="2">
  <div data-grid-item="row-start-2 row-end-4">01</div>
  <div data-grid-item="row-end-3">02</div>
  <div>03</div>
  <div data-grid-item="row-end-4 row-3">04</div>
  <div data-grid-item="row-start-1 row-end-3">05</div>
  <div>06</div>
</div>
```

## レスポンシブ対応

各属性の値を指定する前に、ブレイクポイント名と:を付けることで、ブレイクポイントごとにスタイルを設定できます。

例えば、スマートフォンとPCとでグリッドアイテムの配置場所が異なる時に活用できます。

そのような、ある一定のブラウザサイズから配置場所が変わるような設定も簡単です。



<iframe src="/olayout-docs/demo/grid-item/demo-breakpoint-1.html" style="min-height: 20.5rem"></iframe>


```html "sm:col-3" "sm:row-2" "sm:row-start-1" "sm:row-2" "sm:col-full"
<div data-grid="cols-4 sm:cols-6 sm:row-dense" data-gap="1 sm:2">
    <div data-grid-item="col-full sm:col-3 sm:row-2">01</div>
    <div data-grid-item="col-2">02</div>
    <div data-grid-item="col-2">03</div>
    <div data-grid-item="sm:row-start-1 sm:row-2">04</div>
    <div data-grid-item="col-3 sm:col-full">05</div>
  </div>
```

## 特定の箇所だけの値

例えば、24列のグリッドシステムを使いたい時など、特定の箇所のみに適用したいスタイルがある場合は、カスタプロパティを直接上書きすることで対応できます。

| カスタムプロパティ名 | 値  |
| -------------------- | --- |
| `--ol-grid-col-start` |  `grid-column-start` の値  |
| `--ol-grid-col-end` |   `grid-column-end` の値   |
| `--ol-grid-row-start` |   `grid-row-start` の値   |
| `--ol-grid-row-end` |   `grid-row-end` の値   |


### 手軽に任意の値を使う

手軽な方法は、`style` 属性を使ってカスタムプロパティを上書きすることです。



<iframe src="/olayout-docs/demo/grid-item/demo-custom-1.html" style="min-height: 20.5rem;"></iframe>


```html "--ol-grid-col-start: 2;" "--ol-grid-col-end: 24" "--ol-grid-col-start: span 14" "--ol-grid-col-end: 20" "--ol-grid-col-end: -1"
// index.html
<div data-grid data-gap="2" style="--ol-grid-cols: repeat(24, minmax(0, 1fr))">
  <div data-grid-item="col-full">01</div>
  <div data-grid-item style="--ol-grid-col-start: 2; --ol-grid-col-end: 24">02</div>
  <div data-grid-item style="--ol-grid-col-start: span 14">03</div>
  <div data-grid-item="col-5" style="--ol-grid-col-end: 20">04</div>
  <div data-grid-item="col-3" style="--ol-grid-col-end: -1">05</div>
</div>
```

この方法はブレイクポイントごとに値を変更することができません。

### ブレイクポイントごとの値

ブレイクポイントごとに値を変更したい場合は、その要素に`class`属性かカスタムデータ属性を使って任意の値を指定し、そのセレクターに対してCSSファイル内でメディアクエリを利用して変更してください。


<iframe src="/olayout-docs/demo/grid-item/demo-custom-2.html" style="min-height: 8rem"></iframe>



```html "my-cols"
// index.html
<div data-grid data-gap="2" style="--ol-grid-cols: repeat(16, minmax(0, 1fr))">
  <div data-grid-item="my-cols">01</div>
</div>
```

```css
// style.css
[data-grid-item~="my-cols"] {
  --ol-grid-col-start: 2;
  --ol-grid-col-end: 16;
}

@media (--ol-md) {
  [data-grid-item~="my-cols"] {
    --ol-grid-col-start: 3;
    --ol-grid-col-end: 15;
  }
}

@media (--ol-lg) {
  [data-grid-item~="my-cols"] {
    --ol-grid-col-start: 4;
    --ol-grid-col-end: 14;
  }
}
```
