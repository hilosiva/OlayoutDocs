ボックスのコンテナーを設定する属性

## ボックスのセンタリング

| 属性名    | 属性値                    | 説明 |
| --------- | ------------------------- | ---- |
| `data-container` | - | 上下の `margin` |

## オフセット
| 属性名    | 属性値                    | 説明 |
| --------- | ------------------------- | ---- |
| `data-offset` | `ブレイクポイント名:small` | 左右のオフセット値 |
| `data-offset` | `ブレイクポイント名:medium` | 左右のオフセット値 |
| `data-offset` | `ブレイクポイント名:large` | 左右のオフセット値 |
| `data-offset` | `ブレイクポイント名:xlarge` | 左右のオフセット値 |
