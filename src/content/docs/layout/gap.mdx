| 属性名    | 属性値                    | 説明 |
| --------- | ------------------------- | ---- |
| `data-gap` | スペース倍率（最大： `20`） | 各アイテムの上下左右の間隔を設定 |
| `data-gap-y` | スペース倍率（最大： `20`） | 各アイテムの上下の間隔を設定 |
| `data-gap-x` | スペース倍率（最大： `20`） | 各アイテムの左右の間隔を設定 |

`data-gap` 属性には、スペース量の基準値に対しての倍率を指定します。

スペース基準値はデフォルトでは `8px` が設定されているため、例えば `data-gap` に `2` を指定すると、 `8px` × `2` の `16px` となります。

これを `rem` に変換した `1rem` がスペース量としてセットされます。
