Olayoutがマークアップ中にできることは、基本的にはレイアウトのみです。<br />コンポーネントのスタイリングはあなた自身のCSSファイルで行って頂く必要があります。
そんなコンポーネントのスタイリングをする際に便利な関数を用意しました。これで、Sassなどのプリプロセッサを使わずともサイズの計算が楽になります。


:::caution
CDN版では利用できません。
:::

---

## fluid()

最小値と最大値を指定すると、推奨値を計算した `clamp()` を作成できます。

基本的な使い方は `fluid(最小値px, 最大値px)` を指定するだけです。

これにより、`olayout.config.cjs` の スマートフォン用デザインカンプ幅と、PC用デザインカンプ幅に元に、推奨値を計算してくれます。

```css "fluid(12, 20)"
// style.css
.text {
  /* fluid(最小値px, 最大値px) */
  font-size: fluid(12, 20);  /* clamp(0.75rem, 0.607rem + 0.71vw, 1.25rem) */
}

```

`fluid(最小値px, 最大値px, 最小ビューポート、最大ビューポート)` とすることで、自分で最小ビューポートと最大ビューポートを指定する事もできます。

```css "fluid(12, 20, 640, 1920)"
// style.css
.text {
  /* fluid(最小値px, 最大値px, 最小ビューポート、最大ビューポート) */
  font-size: fluid(12, 20, 640, 1920);  /* clamp(0.75rem, 0.5rem + 0.63vw, 1.25rem) */
}

```


## rem()

pxをremに変換する関数です。

`calc()` を使っても簡単に変換できますが、関数があるとやはり楽です。

使い方は、`rem()` の `()` 内に remに変換したいpx値を入れるだけです。


```css "rem(20)"
// style.css
.text {
  /* rem(px値) */
  font-size: rem(20); /* 1.25rem */
}
```



## em()

pxをemに変換する関数です。

`calc()` を使っても簡単に変換できますが、これも関数があると楽です。

使い方は、`em()` の `()` 内に remに変換したいpx値と、ベースの文字サイズpxを入れます。


```css "em(20, 12)"
// style.css
.text {
  /* rem(px値, ベースの文字サイズpx) */
  font-size: rem(20, 12); /* 1.66666666667em */
}
```



## vw()

pxをvwに変換する関数です。
使い方は、`vw()` の `()` 内に vwに変換したいpx値を入れると、画面幅に応じてデザインカンプの幅をビューポートとしたvw値に変換します。


```css "vw(20)"
// style.css
.text {
  /* vw(px値) */
  font-size: vw(20);
}
```

第二引数にベースとなるビューポートを指定することもできます。


```css "vw(20, 1920)"
// style.css
.text {
  /* vw(px値, ビューポート) */
  font-size: vw(20, 1920);
}
```

これらを活用して、コンポーネントのスタイリングを楽しんでください。
