---
title: Media Queries
description: A guide in my new Starlight docs site.
---

OlayoutCSSでは、olayout.config.cjs の screens のサイズに合わせてカスタムメディアクエリを利用できます。


```css
@media (--ol-sm) {
 /* デフォルトでは screen and (min-width: 576px) と同じ */
}

@media (--ol-md) {
 /* デフォルトでは screen and (min-width: 768px) と同じ */
}

@media (--ol-lg) {
 /* デフォルトでは screen and (min-width: 1024px) と同じ */
}
```
