[フレックスアイテム](../flex-item/) や[グリッドアイテム](../grid-item/)に `data-order`属性を使えば、フレックスアイテムやグリッドアイテムの順番を変更することができます。


| 属性値         | 説明                                 | 適用箇所     |
| -------------- | ------------------------------------ | -------- |
| `0` ~ `20`     | パディングに指定するするスペース倍率 | 上下左右 |
| `y-0` ~ `y-20` | パディングに指定するするスペース倍率 | 上下     |
| `x-0` ~ `x-20` | パディングに指定するするスペース倍率 | 左右     |
| `t-0` ~ `t-20` | パディングに指定するするスペース倍率 | 上       |
| `r-0` ~ `r-20` | パディングに指定するするスペース倍率 | 右       |
| `b-0` ~ `b-20` | パディングに指定するするスペース倍率 | 下       |
| `l-0` ~ `l-20` | パディングに指定するするスペース倍率 | 左 |

---

## 基本的な使い方



`data-p`属性を使用して、マージン（間隔）を変更できます。


<iframe src="/olayout-docs/demo/padding/demo-1.html" style="min-height: 120px"></iframe>



```html "data-p="y-6""
// index.html
<div data-p="y-6">
  ...
</div>
```
