## 基本的な使い方


`data-m`属性を使用して、マージン（間隔）を変更できます。


<iframe src="/olayout-docs/demo/margin/demo-1.html" style="min-height: 120px"></iframe>



```html "data-m="y-6""
// index.html
<div data-m="y-6">
  ...
</div>
```
