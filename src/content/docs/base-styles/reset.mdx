Olayout CSSは、ブラウザの初期スタイルをリセットします。
リセットには、俺流のリセットCSSである、[Oreset.css](https://github.com/hilosiva/oreset.css) をOlayout用にカスタムして利用しています。

## 特徴
- 全ての要素は `box-sizing: border-box`
- 不必要な `margin` 、 `padding、` `border` を排除
- 文字サイズのリセットは行わない
- `:where()` を使って詳細度を限りなく低くキープ
- `font-feature-settings` による文字詰め
- `line-height`は `1.5` をデフォルトにする
- `html` 要素に `scroll-behavior: smooth` を指定
- アクセシビリティに考慮（視差効果など）



---

## 余白のリセットと `border-box`

Olayout CSSは、すべての要素の `margin` と `padding` をリセットします。
また、`box-sizing` も `border-box` に設定します。

```css
*,
::before,
::after {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
```


## 文字サイズのリセットは行わない

Olayout CSSでは、文字サイズのリセットは行いません。
これによりスタイルを指定し忘れた見出しであっても、見出しが通常のテキストのように見えることを防げます。


```css
:where(p, h1, h2, h3, h4, h5, h6) {
  overflow-wrap: break-word;
}
```



## `:where()` を使って詳細度を限りなく低くキープ
リセットのCSSは、他のすべてのスタイルがカスケードできるように詳細度は低くキープしておくほうがよかったります。

Olayout CSSでは、`:where()` を使って詳細度が上がらない工夫をしています。


```css ":where(html)"
:where(html) {
  block-size: 100%;
  scroll-behavior: var(--#{$prefix}-scroll-behavior);
  -webkit-text-size-adjust: none;
}
```



## `font-feature-settings` による文字詰め

Webでも文字詰めはこだわりたいですよね。

Olayoutでは、デフォルトでは `font-feature-settings: "pkna"` を使ってプロポーショナルメトリクスによる文字詰めを行います。

変更する場合は、[文字関連のカスタマイズに関するドキュメント](/olayout-docs/guides/configuration/#typos)を参照してください。

```css {8}
:where(body) {
  min-block-size: 100%;
  text-rendering: optimizeLegibility;
  color: var(--#{$prefix}-text-color-1);
  background-color: var(--#{$prefix}-base-color);
  line-height: var(--#{$prefix}-base-line-height);
  font-family: var(--#{$prefix}-font-set-1);
  font-feature-settings: var(--#{$prefix}-base-feature-settings);
  -webkit-font-smoothing: antialiased;
}

```


# line-hight のデフォルトは1.5

デフォルトの行の高さは、アクセシビリティや可読性を考慮して、1.5 に設定しています。

変更する場合は、[文字関連のカスタマイズに関するドキュメント](/olayout-docs/guides/configuration/#typos)を参照してください。


# `html` 要素に `scroll-behavior: smooth` を指定

html要素には、デフォルトで`scroll-behavior: smooth`を指定していらため、ページ内リンクなどでは、シュルシュルとアニメーションで移動します。

変更する場合は、[アニメーションのカスタマイズに関するドキュメント](/olayout-docs/guides/configuration/# animations)を参照してください。



# アクセシビリティに考慮

OSの視差効果を軽減するオプションを有効にしている場合には、CSSにアニメーションを無効にしたり、フォーカス時のフォーカスリングは削除しないなど、アクセシビリティにも、考慮しています。



おまけとして、form要素の外で、type属性を指定してないbutton要素を使っている場合、そのボタンに警告が表示されるようにもしています。



```css
  # Global
---------------------------------------------- */
*,
::before,
::after {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}

/*
  # Document & Sections
---------------------------------------------- */
:where(html) {
  block-size: 100%;
  scroll-behavior: smooth;
  -webkit-text-size-adjust: none;
}

:where(body) {
  min-block-size: 100%;
  text-rendering: optimizeLegibility;
  line-height: 1.5;
  font-family: system-ui, sans-serif;
  font-feature-settings: "pkna";
  -webkit-font-smoothing: antialiased;
}

/*
  # Grouping content
---------------------------------------------- */
:where(p, h1, h2, h3, h4, h5, h6) {
  overflow-wrap: break-word;
}

:where(hr) {
  border: none;
  border-block-start: 1px solid;
  color: inherit;
  block-size: 0;
  overflow: visible;
}

:where(ul, ol) {
  list-style: none;
}

/*
  # Text-level semantics
---------------------------------------------- */
:where(a:not([class])) {
  -webkit-text-decoration-skip: ink;
          text-decoration-skip-ink: auto;
}

/*
  # Embedded content
---------------------------------------------- */
:where(img, svg, video, canvas, audio, iframe, embed, object) {
  display: block;
}

:where(img, picture, svg) {
  max-inline-size: 100%;
  block-size: auto;
}

:where(svg:not([fill])) {
  fill: currentColor;
}

/*
  # Tabular data
---------------------------------------------- */
:where(table) {
  border-collapse: collapse;
}

/*
  # Forms
---------------------------------------------- */
:where(input, button, textarea, select, optgroup) {
  font: inherit;
  color: inherit;
}

:where(textarea) {
  resize: vertical;
  resize: block;
}

:where(button, label, select, summary, [role=button], [role=option]) {
  cursor: pointer;
}

:where(input[type=file])::-webkit-file-upload-button,
:where(input[type=file])::file-selector-button {
  cursor: pointer;
}

:where(:disabled) {
  cursor: not-allowed;
}

:where(label:has(> input:disabled), label:has(+ input:disabled)) {
  cursor: not-allowed;
}

:where(button) {
  border-style: solid;
}

/*
  # Interactive & Misc
---------------------------------------------- */
:where(template, [hidden]:not([hidden=until-found])) {
  display: none;
}

/*
  # Accessibility
---------------------------------------------- */
:where([disabled], [aria-disabled=true]) {
  cursor: not-allowed;
}

:where([aria-busy=true]) {
  cursor: progress;
}

:where([aria-controls]) {
  cursor: pointer;
}

:where([aria-hidden=false][hidden]) {
  display: initial;
}

:where([aria-hidden=false][hidden]):not(:focus) {
  position: absolute;
  clip: rect(0, 0, 0, 0);
}

:where(:focus:not(:focus-visible), ::before:focus:not(:focus-visible), ::after:focus:not(:focus-visible)) {
  outline: none;
}

/**
 * Prettier ignore
 */
@media (prefers-reduced-motion: reduce) {
  *,
  *:before,
  *:after {
    transition: none !important;
    animation: none !important;
    scroll-behavior: auto !important;
  }
}
```
